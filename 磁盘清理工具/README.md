# 智能磁盘清理工具 SmartDiskCleaner

## 简介
一款像 WizTree 一样扫描磁盘空间，但更智能的磁盘清理工具。
自动识别哪些文件可以安全删除，哪些是系统文件不能动。

## 功能特点
- 🚀 **快速扫描** - 快速遍历目录，显示文件大小和占用空间
- 🧠 **智能分类** - 自动将文件分为四类：
  - 🟢 **安全可删** - 临时文件、缓存、日志、回收站等
  - 🟡 **谨慎操作** - 可能不需要但需确认
  - 🔴 **系统文件** - 自动保护，不可删除
  - ⚪ **未知类型** - 请自行判断
- 🗑️ **一键清理** - 选中安全文件，一键删除释放空间
- 🔍 **灵活筛选** - 按分类、类型筛选，按大小/名称排序
- 📊 **详细统计** - 直观展示空间占用分布

## 安装要求
- 操作系统：Windows 7/10/11
- Python 3.6 或更高版本
- 无需安装第三方库（使用内置 tkinter）

## 使用方法
1. 双击 `启动磁盘清理工具.bat`
2. 选择要扫描的磁盘或目录（如 C:\）
3. 点击「开始扫描」
4. 查看分类结果，勾选要清理的文件
5. 点击「删除选中的安全文件」或「删除所有安全文件」

## 安全建议
- ✅ 优先清理标记为"安全可删"的文件
- ⚠️ 橙色标记的文件建议先查看内容再删除
- ❌ 红色标记的系统文件请勿删除
- 💡 建议定期清理：系统临时文件、浏览器缓存、回收站
